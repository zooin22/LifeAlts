# 삶의 부캐 (LIFE Alts) — 기획 패키지 (확정본)

> 내가 사는 하루하루가 모여, 내 캐릭터의 직업과 정체성이 되는 자기계발 RPG

순수 기획 문서 패키지. 개발은 이 문서를 보고 진행(코드는 새로 작성).

## 한눈에

- **컨셉**: 오늘 한 일 → 캐릭터 성장 → 삶에 따라 직업이 흐르듯 변함 (위치 없음)
- **입력**: 루틴 출석 체크(도장) + 스트릭. 만들 때 스탯 1회 지정(길드 의뢰소 연출).
- **직업**: 정통 판타지 26종. 14일 윈도우 + 70% 규칙 판정. actions에서 매번 파생 계산.
- **퀘스트**: 받기-수행-완수 RPG 사이클. 메인(조정불가/포기가능/마일스톤 보상) + 데일리.
- **캐릭터**: 단순 2D 얼굴 + 직업 의상 레이어(페이퍼돌). 직업별 말투.
- **화면**: RPG HUD — 캐릭터 중앙 + 열리는 창들(퀘스트/스탯/의뢰소/연대기).
- **세계**: 시간대에 따라 배경 변화 (아침·낮·노을·밤). 밤하늘 고정 아님.
- **톤**: 자기계발, "평가하지 않는다". 몬스터 전투 없음(성장/잠듦).
- **이탈 방지**: 채찍은 하는 사람에게, 환대는 돌아온 사람에게.
- **기술**: React Native + TypeScript, Expo. 1단계 로컬(서버 없음).

## 기획 상태

- 1단계 기준 확정. 시스템 뼈대(스탯 6+금화, 직업=파생계산, 스키마 확장 자리) 고정.
- 콘텐츠·기능(직업·의상·문구·금화소비처·경험도감·친구·위젯) 확장 자유.

## 색/비주얼 메모

- 별도 코드 파일 없음. 색은 기획/개발문서에 명시(예: 금색 강조 #F0C070 계열).
- 배경은 시간대별 색 세트로 — 고정 팔레트 아님. 개발 시 토큰화 권장.

## 착수 전

- 권장: 컨셉을 주변에 설명해 반응 검증.
- 가벼운 결정: 시작 직업 수, 캐릭터 베이스 1종, 태그 초기 세트, 판정 출발 수치, 시간대 구간/색.
- 사용자 확인: 앱 이름 상표·앱스토어 중복 / 라이브러리·OS 최신 정책.
