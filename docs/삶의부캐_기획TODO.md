# 삶의 부캐 (LIFE Alts) — 기획 진행 현황 & 남은 일

> 기획서(15장)에 핵심은 다 반영됨. 이 문서는 "무엇이 정해졌고 / 무엇이 남았나"를 한눈에.
> 표시: ✅ 정해짐(기획서 참조) / 🟡 착수 전 가벼운 결정 / 🟢 나중·확장

---

## ✅ 정해진 것 (기획서에 반영 완료)

- **앱 이름**: 삶의 부캐 / LIFE Alts
- **컨셉/정체성**: 오늘 한 일 → 캐릭터 성장 → 삶에 따라 직업이 변함 (기획서 1~3장)
- **톤**: 자기계발(동기부여), "평가하지 않는다", 같은 편이 건네는 말 (4장)
- **행동 입력**: 루틴 출석 체크(도장) + 스트릭. 추천 목록(스탯 미리 박힘) + 직접 만들 때 스탯 1회 지정. 경험은 스탯 아닌 도감(2단계) (5장)
- **스탯**: 6개(힘·지능·매력·손재주·정신·건강) + 금화. 고정 (6장)
- **업무 원칙**: 업무는 기본값, 일 외 시간이 직업을 가른다 (6장)
- **직업 체계**: 정통 판타지 26종(단일7·조합14·숨은5). 의상은 단계적 (7장)
- **직업 판정**: 14일 윈도우 + 감쇠 + 70% 규칙(단일/조합) + 금화 별도. actions에서 파생 계산 (7장)
- **직업 전환 연출**: 감지→각성→의미→기록 4단계 (7장)
- **퀘스트**: 받기-수행-완수 RPG 사이클. 메인(자유 설정·조정불가·포기가능·마일스톤 보상) + 데일리. 데일리로 강제분할 안 함 (8장)
- **몬스터 전투 없음**: 완수=성장 / 미완수=스탯 잠듦 (8장)
- **온보딩**: 4단계, 첫 5분에 첫 도장 체험 (10장)
- **캐릭터 보이스**: 5상황 × 직업별 말투 (11장)
- **화면**: RPG HUD(캐릭터 중앙 + 열리는 창) + 길드 의뢰소(설정) + 모험 연대기(달력) (12장)
- **살아있는 세계**: 시간대에 따라 배경 변화(아침·낮·노을·밤) (12장)
- **이탈 방지**: 며칠 안 함→그리움 / 스트릭 끊김→프리즈·최고기록 박제 / 복귀→환영. "채찍은 하는 사람에게, 환대는 돌아온 사람에게" (13장)
- **기술 스택**: React Native + TypeScript, Expo, 로컬 (개발문서)
- **캐릭터**: 단순 2D 얼굴 + 직업 의상 레이어(페이퍼돌). 2D 확정(3D 비권장)

## 🟡 착수 전 가벼운 결정 (콘텐츠 차원)
큰 틀은 정해졌고, 구체 수치·초기 세트만 정하면 됨.

- **1단계 직업 수**: 26종 중 몇 개부터 의상 만들지 (단일 7종 우선 권장)
- **캐릭터 베이스 1종**: 단순 2D 얼굴/몸 최종 디자인 확정
- **행동 태그 추천 목록 초기 세트**: 운동·독서·물 등 + 각 무게(amount)
- **직업 판정 출발 수치**: 14일 / 감쇠 ×0.9 / 70% — (출시 후 튜닝 전제의 시작값)
- **시간대 구간·색 세트**: 아침/낮/노을/밤의 시각 경계와 색
- **데일리 퀘스트 하루 몇 개가 적당한지**
- **캐릭터 보이스 빈도**: 얼마나 자주 말 걸지 (과하면 피로)

## 🟢 나중 / 확장 (2단계+)
시스템 뼈대를 안 건드리고 추가 가능 (기획서 15장 확장 원칙).

- **금화 소비처 / 인벤토리**: 꾸미기(의상·테마), 스트릭 복구권, 숨은 직업 도전권
- **직업 대폭 확장**: 조합·숨은 직업 전체, 전직/티어(견습→전사→검성)
- **경험 도감 / 칭호**: 일회성 경험(영화·여행)을 스탯 아닌 수집으로. 빈도 기반 "경험→스탯 승격"
- **친구 / 공유**: 직업 자랑, 함께 퀘스트. 경쟁은 검증 문제라 신중. 서버 필요
- **검증 강화**: 친구 들어오면 기기 연동(만보계)·타이머 등 검토
- **수익화**: 프리미엄 의상·테마·구독. 바이럴/정착 후
- **워라밸 톤 장치**: "금화만 많고 정신·건강 0이면 지친 상인" 균형 권장
- **위젯**: 스트릭 강조 위젯. 앱 본체 후 추가
- **추가 테마·날씨·계절**: 시간대 세계의 확장

## ⭐ 개발 전 권장 (검증)
- 컨셉을 주변 사람에게 말로 설명 → 반응 보기. ("오늘 한 일로 캐릭터 직업이 바뀌는 자기계발 앱 어때?")
- "나라면 이걸 매일 쓸까?"에 솔직한 답.
- 앱 이름 "삶의 부캐 / LIFE Alts" 상표·앱스토어 중복 확인 (직접).
- 라이브러리·Expo·OS 정책은 착수 시점 최신 확인.
